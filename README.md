#  TrustLayer — AI Financial Fraud Detection System

TrustLayer is a real-time AI-powered system that detects financial fraud, scams, and risky transactions in chat messages using Gemini AI and community-driven intelligence.

It is designed to help users identify scams before they lose money by analyzing message patterns, urgency, and suspicious financial behavior.

---

##  Features

-  Real-time fraud detection using AI (Gemini)
-  Risk scoring system (0–100)
-  Smart explanation of scam patterns
-  Community-based fraud signal detection
-  Simple analytics dashboard (Streamlit)
-  Telegram bot integration for instant analysis

---

##  Problem It Solves

Financial scams are increasing rapidly through:
- Fake investment schemes
- Urgent money requests
- Phishing links
- Social engineering attacks

TrustLayer helps users detect these threats **before they act**, reducing financial loss and improving digital safety.

---

##  Tech Stack

- Python 
- Google Gemini AI

---

##  How It Works

1. User sends a message (Telegram or input system)
2. AI analyzes the text for fraud patterns
3. System generates:
   - Risk Score
   - Category (Safe / Suspicious / Scam)
   - Explanation
4. Community layer checks for repeated scam patterns
5. Result is returned instantly
